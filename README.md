# Portfolio

Este é um projeto de portfólio pessoal desenvolvido com HTML e CSS avançados utilizando como base o figma fornecido pela RocketSeat. O site apresenta informações sobre mim, meus projetos e habilidades.

## Visualização

O projeto está hospedado no GitHub Pages e pode ser visualizado diretamente no navegador através do seguinte link:

[https://vitorcavic.github.io/portfolio](https://vitorcavic.github.io/portfolio)

## Como executar localmente

Para executar o projeto localmente em sua máquina, siga os passos abaixo:

1. **Clone o repositório:**
   ```
   git clone https://github.com/Vitorcavic/portfolio.git
   ```

2. **Navegue até o diretório do projeto:**
   ```
   cd portfolio
   ```

3. **Abra o arquivo `index.html` no navegador:**
   - Clique duas vezes no arquivo `index.html` ou
   - Abra seu navegador favorito e arraste o arquivo `index.html` para a janela do navegador.

O site será carregado e você poderá visualizar todas as seções do portfólio.

## Estrutura do projeto

- `index.html`: Arquivo principal do site
- `assets/`: Pasta com imagens e logos
  - `images/`: Imagens do projeto
  - `logos/`: Logos utilizados
- `styles/`: Arquivos de estilo CSS
  - `global.css`: Estilos globais
  - `index.css`: Arquivo onde estão os imports dos arquivos css
  - `sections.css`: Estilos das seções

## Tecnologias utilizadas

- HTML5
- CSS3
